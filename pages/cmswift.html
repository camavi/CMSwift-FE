Core

CMSwift e forte quando vuoi scrivere poco, ma restare vicino al DOM.

Renderer dichiarativo, reactive core e component model stanno nella stessa linea: pochi concetti, composizione diretta, meno astrazione inutile.

Punti forti

  • Helper HTML leggibili
  • `signal`, `computed`, `effect`, `batch`
  • Mount e component model senza overhead grosso

Dove aiuta davvero

  • Pagine backoffice
  • Dashboard reattive
  • UI custom dove vuoi controllo pieno

Esempio

const [count, setCount] = _.signal(0);
const doubled = _.computed(() => count() * 2);

_.button({ onClick: () => setCount(count() + 1) }, "Add");
